Lab Dice

This is a simple tool to generate tables that allocate experimental subjects to groups and to randomise the order of experimental blocks (or conditions, trials, etc). It is not yet at version 1.0, and while it is useable it is unstable. Fill in the number and press "Roll!" to get your table. Print you table if you like, saving is coming in the future.

Installation

Get the latest release binary from https://github.com/Tanglo/LabDice/releases and put the .app file where you want to run it from
